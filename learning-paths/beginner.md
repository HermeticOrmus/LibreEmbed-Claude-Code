# Beginner Learning Path - Embedded Systems

## Overview

This beginner path covers Embedded Systems concepts and practices, progressing from fundamentals to practical application with Claude Code plugins.

## Prerequisites

- C programming basics\n- Basic electronics knowledge\n- Understanding of binary and hex

## Modules

### Module 1: Embedded Fundamentals

#### Concepts
- Microcontroller architecture\n- Memory-mapped I/O\n- GPIO programming\n- Clock configuration\n- Build toolchains (GCC ARM)

#### Hands-On Exercise
Apply the concepts using the relevant plugins. Start simple, build complexity gradually.

#### Key Takeaways
- Understand the fundamentals of Embedded Fundamentals
- Know when to apply these patterns
- Create basic implementations confidently

---

### Module 2: First Embedded Project

#### Concepts
- Setting up a development board\n- Blinky LED program\n- UART serial communication\n- Button input with debouncing\n- Basic timer usage

#### Hands-On Exercise
Build on Module 1 with more advanced techniques. Focus on quality and reproducibility.

#### Key Takeaways
- Integrate multiple concepts effectively
- Apply established best practices
- Evaluate and improve existing work

---

### Module 3: Communication Protocols

#### Concepts
- UART in depth\n- SPI master/slave\n- I2C addressing and transactions\n- Protocol debugging with logic analyzer\n- Choosing the right protocol

#### Hands-On Exercise
Combine everything into a comprehensive project demonstrating full mastery.

#### Key Takeaways
- Synthesize all module concepts
- Make informed architectural decisions
- Create production-quality implementations

## Assessment

- [ ] Complete all module exercises
- [ ] Build a project combining all concepts
- [ ] Review against skill file patterns
- [ ] Identify areas for deeper study

## Next Steps

Continue to **intermediate** for RTOS concepts, interrupts, DMA, and more.
