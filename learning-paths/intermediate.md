# Intermediate Learning Path - Embedded Systems

## Overview

This intermediate path covers Embedded Systems concepts and practices, progressing from fundamentals to practical application with Claude Code plugins.

## Prerequisites

- Completed beginner path\n- Working with a dev board\n- C programming proficiency

## Modules

### Module 1: RTOS Fundamentals

#### Concepts
- Why use an RTOS\n- FreeRTOS task creation\n- Priority and scheduling\n- Queues and semaphores\n- Mutexes and deadlock prevention

#### Hands-On Exercise
Apply the concepts using the relevant plugins. Start simple, build complexity gradually.

#### Key Takeaways
- Understand the fundamentals of RTOS Fundamentals
- Know when to apply these patterns
- Create basic implementations confidently

---

### Module 2: Interrupts & DMA

#### Concepts
- Interrupt service routines (ISR)\n- NVIC priority configuration\n- DMA transfer setup\n- ISR best practices\n- Interrupt-driven vs polling

#### Hands-On Exercise
Build on Module 1 with more advanced techniques. Focus on quality and reproducibility.

#### Key Takeaways
- Integrate multiple concepts effectively
- Apply established best practices
- Evaluate and improve existing work

---

### Module 3: Sensor Integration

#### Concepts
- ADC configuration\n- Sensor driver development\n- Calibration techniques\n- Digital filtering (moving average, Kalman)\n- Sensor fusion basics

#### Hands-On Exercise
Combine everything into a comprehensive project demonstrating full mastery.

#### Key Takeaways
- Synthesize all module concepts
- Make informed architectural decisions
- Create production-quality implementations

## Assessment

- [ ] Complete all module exercises
- [ ] Build a project combining all concepts
- [ ] Review against skill file patterns
- [ ] Identify areas for deeper study

## Next Steps

Continue to **advanced** for safety-critical systems, FPGA, and custom bootloaders.
