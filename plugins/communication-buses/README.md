# Communication Buses

I2C, SPI, UART, CAN, USB protocols and drivers

## What's Included

### Agents
- **Bus Engineer** - Specialized agent for I2C, SPI, UART, CAN, USB protocols and drivers

### Commands
- `/comm-bus` - Quick-access command for communication-buses workflows

### Skills
- **Comm Bus Patterns** - Pattern library and knowledge base for communication-buses

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/comm-bus analyze

# Use the command with specific input
/comm-bus generate --context "your project"

# Reference patterns from the skill
"Apply comm-bus-patterns patterns to this implementation"
```

- Follow established conventions for communication-buses
