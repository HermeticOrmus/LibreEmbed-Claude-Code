# Iot Protocols

MQTT, CoAP, LwM2M, BLE, LoRaWAN, Zigbee

## What's Included

### Agents
- **Iot Protocol Engineer** - Specialized agent for MQTT, CoAP, LwM2M, BLE, LoRaWAN, Zigbee

### Commands
- `/iot` - Quick-access command for iot-protocols workflows

### Skills
- **Iot Protocol Patterns** - Pattern library and knowledge base for iot-protocols

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/iot analyze

# Use the command with specific input
/iot generate --context "your project"

# Reference patterns from the skill
"Apply iot-protocol-patterns patterns to this implementation"
```
